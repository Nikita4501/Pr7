# Отладка приложения «Фибоначчи»

В процессе отладки были использованы следующие средства Visual Studio:
- **Точки останова** для приостановки выполнения на интересующей строке.
- **Пошаговое выполнение** (F10, F11) для детального анализа работы цикла.
- **Окно «Локальные»** для отслеживания значений переменных `n`, `n1`, `n2`, `sum`.
- **Анализ стека вызовов** для понимания последовательности вызовов методов.
